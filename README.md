# OrangeHRM — Login Automation Testing with Selenium & Java

## 📌 Project Overview
Automated testing project for OrangeHRM web application using 
Selenium WebDriver and Java, focusing on Login functionality 
and authentication scenarios.

## 🛠️ Tools & Technologies
- Java
- Selenium WebDriver
- Maven (pom.xml)
- IntelliJ IDEA

## 🎯 Test Scenarios
- Valid login with correct credentials
- Invalid login with wrong password
- Invalid login with wrong username
- Empty fields validation
- Login page UI verification

## 📁 Project Structure


## ▶️ How to Run
1. Clone the repository
2. Open project in IntelliJ IDEA
3. Run `mvn install` to install dependencies
4. Run the test classes

## 🔗 Application Under Test
- **App:** OrangeHRM (https://opensource-demo.orangehrmlive.com)
- **Type:** Web Application — HR Management System
