# OrangeHRM — Login Automation Testing with Selenium & Java

## 📌 Project Overview
Automated testing project for OrangeHRM web application using 
Selenium WebDriver and Java, focusing on Login functionality 
and authentication scenarios.

## 🛠️ Tools & Technologies
- Java
- Selenium WebDriver
- Maven (pom.xml)
- IntelliJ IDEA

## 🎯 Test Scenarios
- Valid login with correct credentials
- Invalid login with wrong password
- Invalid login with wrong username
- Empty fields validation
- Login page UI verification
